# 功能规划路线图

本文档记录 Obsidian Language Learner 插件的未来开发计划和功能增强建议。

---

## 一、核心功能增强

### 1.1 内置间隔复习系统

**优先级：高**

**现状：** 目前插件依赖 AnkiConnect 导出到 Anki 进行复习，用户需要额外安装 Anki。

**规划：**
- 实现内置 SM-2 算法或其他 SRS 算法
- 添加复习面板，支持正反面卡片复习
- 记录复习历史和正确率
- 支持自定义复习间隔参数

**实现思路：**
- 在数据库中添加复习相关字段（下次复习时间、复习次数、难度系数）
- 新增复习视图（ReviewView）
- 使用 Obsidian 定时任务提醒复习

**预期效果：** 用户无需依赖 Anki 即可完成完整的单词学习和复习流程。

---

### 1.2 发音功能增强

**优先级：中**

**现状：** 依赖有道等在线词典的发音，功能有限。

**规划：**
- 集成多源 TTS API（有道、百度、Google TTS）
- 设置中可选择美式/英式发音偏好
- 在查词面板和复习面板显示音标
- 支持语速调节

**实现思路：**
- 封装统一 TTS 接口，支持多源切换
- 从词典数据中提取音标显示
- 添加发音缓存机制，减少网络请求

---

### 1.3 文本分析工具

**优先级：中**

**现状：** 阅读模式可展示文章，但缺乏分析功能。

**规划：**
- 分析文章生词密度和难度等级
- 按词频排序展示生词列表
- 一键导入文章中所有生词到词库
- 生成文章词汇统计报告

**实现思路：**
- 集成词频数据库（如 COCA 词频表）
- 在阅读模式侧边栏显示词汇分析面板
- 添加批量导入命令

---

## 二、用户体验优化

### 2.1 快捷键系统

**优先级：中**

**现状：** 仅支持鼠标划词查词。

**规划：**
- 自定义快捷键配置界面
- 常用操作快捷键支持：
  - 快速查词（输入模式）
  - 切换单词状态（学习→熟悉→掌握→会了）
  - 在列表中导航（上/下一个单词）
  - 快速添加例句/笔记

**实现思路：**
- 使用 Obsidian 的 `addCommand` API 注册快捷键命令
- 设置页面添加快捷键自定义区域
- 支持 Vim 风格快捷键映射

---

### 2.2 移动端优化

**优先级：高**

**现状：** 刚修复 Android 端加载问题，但移动端体验仍有优化空间。

**规划：**
- 优化触摸操作响应
- 添加手势支持：
  - 左右滑动切换单词
  - 下拉刷新列表
  - 长按显示详情
- 适配移动端界面布局
- 优化虚拟键盘交互

**实现思路：**
- 使用 Obsidian 的 `Platform.isMobile` 判断平台
- 引入触摸事件监听
- 调整移动端 CSS 布局

---

### 2.3 批量操作功能

**优先级：中**

**现状：** 仅支持单个单词操作。

**规划：**
- 多选模式批量操作
- 支持的批量操作：
  - 修改状态（学习/熟悉/掌握/会了/忽略）
  - 添加/移除标签
  - 导出选中单词
  - 删除选中单词
- 全选/反选功能

**实现思路：**
- 在单词列表添加多选模式切换按钮
- 使用 checkbox 实现多选
- 批量操作调用数据库 API

---

## 三、数据与同步

### 3.1 多源数据导入

**优先级：中**

**现状：** 支持 JSON 导入，来源有限。

**规划：**
- 支持 Kindle 生词本导入（.vocab 文件）
- 支持浏览器插件数据导入
- 支持 CSV/TSV 格式导入
- 支持其他背词软件导出格式（如扇贝、百词斩）

**实现思路：**
- 为每种格式编写解析器
- 添加导入向导，支持字段映射
- 导入前预览数据

---

### 3.2 云同步功能

**优先级：低**（原服务器功能已移除，可作为长期规划）

**现状：** 依赖 Markdown 文件 + Obsidian Sync，或 IndexedDB 本地存储。

**规划：**
- 支持第三方云存储同步（WebDAV、Dropbox、Google Drive）
- 端到端加密保护隐私
- 冲突检测与合并策略
- 离线优先设计

**实现思路：**
- 使用 Web 标准API（如 WebDAV）
- 实现增量同步减少流量
- 本地数据优先，云端备份

---

### 3.3 数据备份与恢复

**优先级：中**

**现状：** 手动导出，无自动备份。

**规划：**
- 自动定时备份（每日/每周）
- 备份文件管理（保留最近 N 个备份）
- 一键恢复到指定备份点
- 备份到指定目录或云存储

**实现思路：**
- 使用 Obsidian 定时任务或 setInterval
- 备份文件命名为 `backup_YYYYMMDD_HHmmss.json`
- 设置中配置备份频率和保留数量

---

## 四、学习功能扩展

### 4.1 生词测试模块

**优先级：中**

**规划：**
- 多种测试模式：
  - 选择题（中文选英文/英文选中文）
  - 填空题
  - 听音拼写
- 按标签/状态筛选测试范围
- 记录测试成绩和正确率统计
- 错题本功能

**实现思路：**
- 新增测试视图（TestView）
- 根据测试结果自动更新单词状态
- 使用图表展示测试成绩趋势

---

### 4.2 单词关联与扩展

**优先级：低**

**规划：**
- 同义词/反义词展示
- 词根词缀分析
- 词族扩展（如 act → action → active → activity）
- 搭配短语推荐

**实现思路：**
- 集成词根词缀数据库
- 从词典数据中提取同义词/反义词
- 在单词详情页展示关联信息

---

## 五、开发者与社区

### 5.1 插件 API

**优先级：低**

**规划：**
- 暴露核心 API 供其他插件调用
- 支持自定义词典源
- 支持自定义复习算法

---

### 5.2 国际化完善

**优先级：中**

**现状：** 支持中英文，部分字符串未翻译。

**规划：**
- 完善所有界面的国际化
- 支持更多语言（日语、韩语等）
- 社区贡献翻译

---
